# Pika Skills

> [!IMPORTANT]
> **This repository is deprecated and no longer maintained.**
>
> The Pika Developer API (`pika.me/dev`) that powered these skills has been discontinued, and the skills in this repository no longer work. We're rebuilding platform-agnostic access to PikaStream 1.0 and later generation models — in the meantime, visit **[experiment.pika.art](https://experiment.pika.art)** or follow us on X at @pika_labs to use Pika's latest models.

## What happened

We're focused on improving our models and revamping how developers and agents access them. As part of this work:

- **The original `pika.me/dev` Developer API has been deprecated.** Existing Developer Keys (`dk_...`) no longer work, and new keys can no longer be created. Email support@pika.art if you have outstanding credits, and we'll migrate you to the new platform when it launches.
- **The skills in this repository are non-functional** as of this change, since they all depend on that API.
- **Previously listed commercial terms no longer apply.** Future programmatic access details will be announced when the new platform launches.

## Where to go instead

👉 **[experiment.pika.art](https://experiment.pika.art)** — try Pika's latest products and models today, and stay tuned about Pikastream model launches.

## What's next

We're building a new, platform-agnostic way to programmatically access PikaStream 1.0 and later generation models. When it's ready, we'll publish:

- Updated API access and documentation
- Updated commercial terms
- Migration guidance for anyone who built on the previous Developer API

Watch [pika.art](https://pika.art) and our official channels for announcements.
